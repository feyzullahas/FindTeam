# FindTeam Frontend

React frontend for finding football teammates.

## Features

- Google OAuth authentication
- User profile management
- Post creation and browsing
- Search and filter functionality
- Responsive design with Tailwind CSS

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- React Router
- React Query
- React Hook Form
- Tailwind CSS
- Lucide React Icons
- Axios

## Environment

Make sure the backend server is running on `http://localhost:8000` for the API calls to work.

## Pages

- `/` - Ana Sayfa
- `/login` - Giriş
- `/profile` - Profil Yönetimi
- `/create-post` - İlan Ver
- `/posts` - İlanları Listele
